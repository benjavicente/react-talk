## En resumen ¿Qué es React?

React es una **librería popular** de JavaScript
simple y **poco _opinionated_** para construir interfaces
de usuario de forma **declarativa**

```tsx
import React from "react";

function App() {
	return React.createElement("div", null, "¡Hola, mundo!");
}
```

Es utilizado en conjunto a **JSX**

```tsx
function App() {
	return <h1>¡Hola, mundo!</h1>;
}
```
