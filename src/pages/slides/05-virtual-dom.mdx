## Virtual DOM

Es una extructura de datos que mantiene
React que indica como debería ser el DOM real.

Cuando se hacen cambios en el virtual, React
**los compara** y aplica los cambios necesarios.

Aquí podemos definir nuestros propios componentes.

```tsx
<App>
	<Header />
	<Main>
		<Sidebar />
		<Content />
	</Main>
	<Footer />
</App>
```

Los componentes se definen con **`CamelCase`**.
