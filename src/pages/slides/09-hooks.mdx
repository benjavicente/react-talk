## Hooks

Los hooks de React son **funciones** que permiten
manejar estado y ciclo de vida de componentes.
Son una aÃ±adiciÃ³n reciente a React, utilizados
en conjunto a los componentes funcionales.

Los principales son `useState` y `useEffect` y `useContext`.
[La API de hooks tiene la lista completa][hooks-api].

[hooks-api]: https://reactjs.org/docs/hooks-reference.html

```tsx
function App() {
	const [count, setCount] = useState(0);
	useEffect(() => {
		document.title = `Apretado ${count} veces`;
	}, [count]);
	return (
		<button onClick={() => setCount(count + 1)}>
			Incrementar el contador
		</button>
	);
}
```
