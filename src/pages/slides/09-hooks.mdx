## Hooks

Los hooks de React son **funciones** que permiten
manejar estado y ciclo de vida de componentes.
Son una añadición reciente a React, utilizados
en conjunto a los componentes funcionales.

Los principales son `useState` (estado) y
`useEffect` (efectos dado cambios) y `useContext` (estado del árbol).

```tsx
function App() {
	const [count, setCount] = useState(0);
	useEffect(() => {
		document.title = `Apretado ${count} veces`;
	}, [count]);
	return (
		<button onClick={() => setCount(count + 1)}>
			Incrementar el contador
		</button>
	);
}
```
