## Para ir cerrando: librerías y herramientas

### Librerías

- **Estilos**: TailwindCSS + DaisyUI
- **Lógica de UI**: HeadlessUI
- **Formularios**: Formik
- **Validación**: Yup
- **Llamadas a API**: Axios, SWR, TanStack Query (react-query)

### Herramientas

- **Linters**: ESLint, Prettier, Typescript (+ pluggins para las librerías y frameworks)
- **React devtools**: Extención para debuggear
- **Hosting**: Cloudflare Pages, Vercel, Netlify, Firebase Hosting, AWS Amplify
- **Integraciones como Backend**: NextAuth, Prisma
- **Servicios de Backend**: Firebase, Supabase, Strapi, PocketBase, AWS Amplify, Hasura
- **Markdown**: MDX, Remark, Rehype, React Markdown
